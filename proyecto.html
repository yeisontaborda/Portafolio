Detalles del Proyecto Saúl

¿Cuándo fue el proyecto?

El proyecto se realizó en el primer semestre de 2023.

¿Para qué fue realizado?

El proyecto fue realizado para mejorar los sistemas de egresados de la universidad, tanto base de datos como otro tipo de usos.

¿Qué se logró?

Aunque no se implementó, se logró crear algo funcional que cumplía con los requerimientos establecidos por la universidad.

¿Cómo se hizo?

Se utilizó un enfoque ágil (Scrum) con reuniones semanales y revisiones constantes del progreso del proyecto.
